STAMP Your Content:
Proving Dataset Membership via Watermarked Rephrasings

[DATA-FM @ ICLR 2025 ]

Saksham Rastogi1, Pratyush Maini2,3, Danish Pruthi1,

▶ Indian Institute of Science, Bengaluru, India ▶ Carnegie Mellon University ▶ DatologyAI
OpenReview ArXiv Code

TLDR

We propose STAMP, a simple method that allows content creators to reliabilty detect if their work was used to train a large language model (e.g., GPT).

Goal: Dataset Inference

  • 1. Large Language Models (LLMs) are trained on vast amounts of text crawled from the internet.
  • 2. We aim to enable content creators to detect if an LLM was trained on their collection of documents.
  • 3. Allows for reliable detection of copyrighted detection and benchmark contaminated in the pretraining corpora.

Method: STAMP


  • 1. Given an original piece of content, STAMP generates multiple rephrased versions, each containing a watermark with a unique secret key. One version is released publicly, while others remain private.
  • 2. Subsequently, creators can compare model likelihoods between public and private versions using paired statistical tests to prove membership.
